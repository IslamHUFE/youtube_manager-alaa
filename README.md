# YouTube Channel Manager

A desktop application built using Python and CustomTkinter to manage YouTube playlists and videos.

## Features
- Create playlists
- Add videos with duration and views
- Save and load playlists from files
- Search playlists
- Modern UI with CustomTkinter

## Requirements
- Python 3.x
- customtkinter==5.2.2

## Install dependencies
```bash
pip install -r requirements.txt
